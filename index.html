Tate's SSP: Download

My Senior Signature Project consists in part of a much-needed upgrade to the Vernier company's Logger Pro software. The objective of this project is to provide, free of charge, software with similar capabilities to their most recent paid graphical analysis tools, as well to as incorporate more up-to-date data analysis tools such as time-series predictive modeling, machine learning, and user-defined scripts. I also hope to make this software straightforward to use, integrate it with cloud-based collaboration tools, and allow for easy modification of its functionality by editing source code (written in Python).


This software is currently an early prototype, with only the most basic elements in place. Please feel free to download and test this proof of concept below.
